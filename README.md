# luery

LJM12914's tool library.

## 基本使用
- 使用`$`或`luery`调用`luery`工具函数集。可通过提前更改`window.lueryShortCut`的值避免`$`冲突。
- 编写`lueryFail()`以自定义不兼容错误的处理。
- 使用`luery.addFunc({customFunc:(arguments)=>{operations;}})`拓展luery。
